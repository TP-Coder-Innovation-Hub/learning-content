# Creating Images with Gemini — From Text to Visual

## What You'll Learn

- How to generate images using Google Gemini
- Prompting techniques for better results
- Practical use cases: presentations, social media, mockups
- Free alternatives and limitations

## Getting Started

### Requirements
- Google account (free)
- Go to [gemini.google.com](https://gemini.google.com)
- Type a description of the image you want

### Basic Prompt

```
Create an image of a modern office workspace with a laptop showing 
a dashboard, a coffee cup, and natural lighting. Photorealistic style.
```

That's it. No special setup, no API, no cost (free tier).

## Prompting for Better Images

### Be Specific About Style

```
Bad:  "Make a picture of a chart"
Good: "Create a minimalist infographic showing 3 growth metrics: 
       Revenue +25%, Users +40%, Retention +15%. 
       Modern design, blue and white color scheme, flat illustration style."
```

### Specify Technical Details

| Parameter | Examples |
|-----------|----------|
| Style | Photorealistic, flat illustration, watercolor, isometric, pixel art |
| Color | Blue and white theme, warm tones, monochrome, neon |
| Composition | Centered, wide angle, close-up, bird's eye view |
| Text | Include text "SALE 50% OFF", no text |
| Aspect ratio | Wide (16:9), square (1:1), portrait (9:16) |

### Prompt Template

```
Create a [style] image of [subject] in [setting]. 
[Specific details about composition, colors, elements]. 
[Mood/lighting]. [Text if any]. [Aspect ratio].
```

## Practical Use Cases

### Use Case 1: Presentation Slides

Prompt:
```
Create a professional title slide background image for a presentation 
about "Digital Transformation 2026". 
Minimalist design, blue gradient background, abstract geometric shapes, 
no text. Wide format 16:9. Clean corporate style.
```

### Use Case 2: Social Media Graphics

Prompt:
```
Create a square Instagram post image for a tech company. 
Show a happy Thai person using a smartphone with a futuristic 
holographic interface. Bright, modern, optimistic mood. 
Flat illustration style. Include text: "AI for Everyone"
```

### Use Case 3: Mockups and Prototypes

Prompt:
```
Create a mockup of a mobile banking app screen showing:
- Account balance at the top
- 3 quick action buttons (Transfer, Pay, Scan)
- Recent transactions list
- Bottom navigation bar with 4 icons

Modern Thai banking app design. Clean white background with blue accents. 
Photorealistic phone frame.
```

### Use Case 4: Blog Post Illustration

Prompt:
```
Create a blog post header image about "Why Developers Should Learn AI". 
Show a developer sitting at a desk with code on one monitor and an AI 
chat interface on another. Warm lighting, modern workspace, 
watercolor illustration style. Wide format.
```

## Thai Text in Images

Gemini (Imagen) supports Thai text in generated images — one of the few AI image tools that does this well.

```
Create a promotional banner for a Thai restaurant. 
Show pad thai and green curry dishes elegantly plated.
Include Thai text: "อร่อยทุกจาน สดใหม่ทุกวัน"
Warm lighting, food photography style, portrait format.
```

## Iteration: When the Image Isn't Right

Don't start over. Refine:

| Problem | Fix |
|---------|-----|
| Wrong style | "Make it more [minimalist/colorful/realistic]" |
| Missing element | "Add [element] to the [position]" |
| Wrong colors | "Change the color scheme to [colors]" |
| Text is garbled | Re-specify the exact text in quotes |
| Wrong layout | "Rearrange so [element] is on the left, [element] on the right" |

## Free Image AI Comparison

| Tool | Free Tier | Thai Text | Best For |
|------|-----------|-----------|----------|
| Gemini (Imagen) | Yes | Good | General purpose, Thai text |
| ChatGPT (DALL-E) | Limited | Poor | Creative illustrations |
| Canva AI | Yes | Good (with editing) | Social media, presentations |
| Leonardo.ai | Limited | Poor | Artistic images |

## Limitations

- Free Gemini has daily generation limits (varies by usage)
- Complex text rendering may still have errors
- Very specific brand guidelines are hard to match exactly
- Cannot generate images of real people (policy restriction)
- Output resolution may not be print-quality

## Key Takeaway

Gemini makes image generation accessible to everyone — no design skills needed. Be specific about style, colors, and composition. Iterate when the first result isn't right. And for Thai-language visuals, Gemini is currently the best free option.
